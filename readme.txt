# A* Vision: Smart Pathfinding System #


INTRODUCTION
Have you ever thought about how Google Maps can find the route so quickly?. How characters in games can move around in the smartest way? The A* algorithm is one of the techniques that makes this happen.
This project is going to explain the A* algorithm in a way so you can understand how to find the best route between two points without wasting time.

WHAT IS A* ALGORITHM?
The A* algorithm is a way to find the best path from one point to another.
It is used in things like:
1.Navigation systems
2.Game development
3.Robotics

HOW IT WORKS?
The A* algorithm uses two main ideas:
1.The actual cost, which is the distance from the start to where you are now.
2.The estimated cost, which is a guess of the distance from where you're now to your goal.
3.It adds these two costs together to get the cost.
4.The algorithm always chooses the path with the total cost.

PROBLEM STATEMENT

The objective of this project is to design and implement a system that finds the shortest path between two nodes in a graph without using Dijkstra’s algorithm, while improving performance using machine learning.          

STEP BY STEP PROCESS
1. You start at the beginning.
2. Then you look at all the paths you can take from there.
3. You calculate the cost for each path.        
4. You choose the path with the cost.
5. You keep doing this until you reach your goal.
